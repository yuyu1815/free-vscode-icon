# vscode-all-in-one-icon

<!-- Plugin description -->

Ein IntelliJ IDEA-Plugin, das schöne VS Code-Stil Datei- und Ordnersymbole in Ihre JetBrains-IDE bringt. Unterstützt mehrere Symbolthemen einschließlich VSCode Icons, Material Icons und File Icons.

**Funktionen:**

- Mehrere Symbolthemen (VSCode Icons, Material Icons, File Icons)
- Themen-Prioritätssystem mit Fallback-Auflösung
- 2.000+ Datei- und Ordnersymbole
- Mehrsprachige Benutzeroberfläche (English, 日本語, 简体中文, 한국어, Deutsch)
- Einfacher Themenwechsel
- Projektebene Konfiguration
<!-- Plugin description end -->

## Funktionen

- **Mehrere Symbolthemen**
  - **VSCode Icons**: Basierend auf dem beliebten [vscode-icons/vscode-icons](https://github.com/vscode-icons/vscode-icons) Projekt
  - **Material Icons**: Material Design inspirierte Symbole
  - **File Icons**: 2.000+ Symbole von [file-icons/vscode](https://github.com/file-icons/vscode)

- **Umfassende Abdeckung**
  - Unterstützung für 100+ Dateierweiterungen
  - Spezielle Symbole für beliebte Frameworks und Tools
  - Benutzerdefinierte Ordnersymbole für gängige Verzeichnisse

- **Einfache Konfiguration**
  - Einfache Einstellungen auf Projektebene
  - Benutzerdefinierte Symbole pro Projekt aktivieren/deaktivieren
  - Themen sofort wechseln ohne Neustart

## Installation

### Über JetBrains Marketplace

1. JetBrains IDE öffnen
2. Gehe zu <kbd>Datei</kbd> > <kbd>Einstellungen</kbd> > <kbd>Plugins</kbd>
3. Klicke auf <kbd>Marketplace</kbd>
4. Suche nach „vscode-all-in-one-icon"
5. Klicke auf <kbd>Installieren</kbd>

### Manuelle Installation

1. Laden Sie die neueste Version von [GitHub Releases](https://github.com/yuyu1815/vscode-all-in-one-icon/releases/latest) herunter
2. Gehe zu <kbd>Datei</kbd> > <kbd>Einstellungen</kbd> > <kbd>Plugins</kbd>
3. Klicke auf das Zahnrad-Symbol ⚙️
4. Wähle <kbd>Plugin von Festplatte installieren...</kbd>
5. Wähle die heruntergeladene Datei

## Konfiguration

Nach der Installation des Plugins:

1. Gehe zu <kbd>Datei</kbd> > <kbd>Einstellungen</kbd> > <kbd>Werkzeuge</kbd> > <kbd>Icon Theme Settings</kbd>
2. **Sprache**: Wähle die gewünschte UI-Sprache
3. **Themen-Priorität**:
   - „Aktive Themen" zeigt Themen in Prioritätsreihenfolge
   - Verwende ▲/▼ Tasten zum Umordnen
   - Verwende Hinzufügen/Entfernen Tasten zur Verwaltung
   - Symbole werden vom obersten Thema aufgelöst, bei Nichtfinden Fallback zum nächsten
4. Aktiviere/deaktiviere „Benutzerdefinierte Dateisymbole aktivieren"
5. Klicke auf <kbd>Übernehmen</kbd> um zu speichern
